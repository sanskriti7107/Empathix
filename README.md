# Empathix

Empathix is an AI-powered empathy and toxicity analysis tool built using Python, Flask, and Hugging Face models.

## Features
- Real-time text analysis
- Empathy and toxicity scoring
- Sentiment-based evaluation
- Machine learning model integration
- API-based text processing

## Technologies Used
- Python
- Flask
- Hugging Face API
- HTML/CSS
- Natural Language Processing (NLP)

## Purpose
The project explores how artificial intelligence can be used to understand communication patterns, identify potentially harmful language, and encourage more empathetic interactions.

## Future Improvements
- Advanced sentiment analysis
- User dashboard and analytics
- Enhanced machine learning models
- Real-time feedback and recommendations
